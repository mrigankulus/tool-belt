Import/Export

Import/Export allows you to easily transfer API Exceptions throughout different environments. Export in one environment (ex. DEV) and import in another (ex. PROD).

Beta! Import/Export is a beta feature that should be used with caution. It currently only supports API Exceptions for the Endpoint-Security-Analyzer.


Are we importing or exporting?

Export

Export will allow you to select items to export, and give you a JSON representation of those items. Copy that JSON to your clipboard, then paste it into the Import section of the environment you'd like to copy to.

Import

If you already have JSON that you've exported from another environment, select import to add those items in this environment.
